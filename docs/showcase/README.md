# Project Showcase

This folder contains quick preview assets for presenting the rewilding platform work in a concise, portfolio-friendly format.

## Suggested framing

- Built a static public-facing geospatial portfolio site around a reproducible national rewilding screening workflow for England.
- Translated raw spatial outputs into a clearer product story: methods, findings, location case studies, and an interactive explorer.
- Added place-based case studies for Cornwall, Somerset, and a northern peatland-borderland zone, using model outputs plus policy and landscape references.
- Packaged the site for lightweight publishing from `docs/`, including local assets, map previews, and downloadable supporting files.
